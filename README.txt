Dependencies Required to Install:


Python is mandatory to run this code as well as the Nltk. 
Please go to this link and install it.

https://www.python.org/downloads/




Or, you can type this in your terminal (assuming you have pip installed on your computer):


pip install python 




The Natural Language Toolkit will also need to be installed. 
Refer to either link to install.


https://www.nltk.org/install.html

https://pythonprogramming.net/tokenizing-words-sentences-nltk-tutorial/



Or, just run this command in your terminal. (Which may install 1.8GB of data)


pip install -U nltk





~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~



To Run:
Navigate to the /cs157a-tokenizer folder via terminal or command prompt. Once you have navigated to inside the folder, run the following command:

py TokenizerClass.py
